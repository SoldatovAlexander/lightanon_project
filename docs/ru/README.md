## Покрытие
Документы описывают текущее состояние проекта:
- пакетная анонимизация (`pandas`/`polars`),
- потоковая обработка событий,
- обратимое обезличивание текста для LLM/RAG,
- инженерные рекомендации по RAG-сценариям с персональными данными.
